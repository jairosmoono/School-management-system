Laravel School Management System

A complete School Management System built with Laravel, designed to manage students, teachers, classes, exams, and administrative operations efficiently.

📌 Features <br>
👨‍🎓 Student Management (admission, profiles, records)
<br>
👩‍🏫 Teacher Management <br>
🏫 Class & Subject Management <br>
🗓️ Timetable Scheduling <br>
📝 Exam & Results Management <br>
💰 Fee Management System <br>
📊 Dashboard with analytics <br>
🔐 User Authentication & Role Management (Admin, Teacher, Student)
 <br> 📁 File Uploads (documents, assignments, etc.)
<br> 🛠️ Built With
Backend: Laravel (PHP Framework)
Frontend: Blade Templates / HTML / CSS / JavaScript
Database: MySQL
Server: Apache / Nginx
Version Control: Git
